Proposed SQL Table Designs for Client Statistics

Table: client_daily_performance_summary

Purpose: To store daily aggregated performance metrics for each client, broken down by client offer and advertising network. This allows for daily trend analysis.

Grain: report_date, client_name, client_offer, ad_network

Column Name Data Type Mode Constraints / Notes Description
report_date DATE NULLABLE PK The date for which the summary is recorded.
client_name STRING NULLABLE PK Name of the client.
client_offer STRING NULLABLE PK Specific offer for the client (e.g., 'Product A', 'All Offers').
ad_network STRING NULLABLE PK Advertising network (e.g., 'Facebook/Meta', 'Google Ads').
total_spend FLOAT NULLABLE Total amount spent.
total_impressions INTEGER NULLABLE Total number of impressions.
total_clicks INTEGER NULLABLE Total number of clicks.
total_reach INTEGER NULLABLE Total unique users reached.
total_purchases INTEGER NULLABLE Total number of purchase conversions.
total_leads INTEGER NULLABLE Total number of lead conversions.
total_registrations INTEGER NULLABLE Total number of registration conversions.
total_conversion_value FLOAT NULLABLE Total monetary value from all conversions.
roas FLOAT NULLABLE Return On Ad Spend (total_conversion_value / total_spend).
cpc FLOAT NULLABLE Cost Per Click (total_spend / total_clicks).
cpm FLOAT NULLABLE Cost Per Mille (Thousand Impressions) (total_spend / total_impressions * 1000).
ctr FLOAT NULLABLE Click-Through Rate (total_clicks / total_impressions).
cpa_purchase FLOAT NULLABLE Cost Per Purchase (total_spend / total_purchases).
cpa_lead FLOAT NULLABLE Cost Per Lead (total_spend / total_leads).
cpa_registration FLOAT NULLABLE Cost Per Registration (total_spend / total_registrations).
frequency FLOAT NULLABLE Average number of times ads were shown to each unique person (total_impressions / total_reach).
active_campaigns_count INTEGER NULLABLE Number of campaigns active on this day.
active_adsets_count INTEGER NULLABLE Number of ad sets active on this day.
active_ads_count INTEGER NULLABLE Number of ads active on this day.
last_updated TIMESTAMP NULLABLE Timestamp of when the record was last updated.

Source: Aggregated daily from v1_b2s_all_acquisition_production_report.

Table: client_campaign_periodic_summary

Purpose: To store periodic (e.g., Last 7 Days, Last 30 Days, Month-to-Date) performance metrics for each campaign associated with a client and offer.

Grain: client_name, client_offer, campaign_id, period_type, period_start_date, period_end_date

Column Name Data Type Mode Constraints / Notes Description
client_name STRING NULLABLE PK Name of the client.
client_offer STRING NULLABLE PK Specific offer for the client.
campaign_id STRING NULLABLE PK Unique identifier for the campaign.
period_type STRING NULLABLE PK Type of period (e.g., 'L7D', 'L30D', 'MTD', 'Custom').
period_start_date DATE NULLABLE PK Start date of the summary period.
period_end_date DATE NULLABLE PK End date of the summary period.
campaign_name STRING NULLABLE Name of the campaign.
ad_network STRING NULLABLE Advertising network.
campaign_status STRING NULLABLE Current status of the campaign (e.g., 'Active', 'Paused').
total_spend FLOAT NULLABLE Total amount spent in the period.
total_impressions INTEGER NULLABLE Total impressions in the period.
total_clicks INTEGER NULLABLE Total clicks in the period.
total_reach INTEGER NULLABLE Total unique users reached in the period.
total_purchases INTEGER NULLABLE Total purchase conversions in the period.
total_leads INTEGER NULLABLE Total lead conversions in the period.
total_registrations INTEGER NULLABLE Total registration conversions in the period.
total_conversion_value FLOAT NULLABLE Total monetary value from conversions in the period.
roas FLOAT NULLABLE ROAS for the period.
cpc FLOAT NULLABLE CPC for the period.
cpm FLOAT NULLABLE CPM for the period.
ctr FLOAT NULLABLE CTR for the period.
cpa_purchase FLOAT NULLABLE Cost Per Purchase for the period.
cpa_lead FLOAT NULLABLE Cost Per Lead for the period.
cpa_registration FLOAT NULLABLE Cost Per Registration for the period.
frequency FLOAT NULLABLE Average frequency for the period.
health_score FLOAT NULLABLE Conceptual campaign health score if calculated.
last_updated TIMESTAMP NULLABLE Timestamp of when the record was last updated.

Source: Aggregated from v1_b2s_all_acquisition_production_report over the specified period.

Table: client_overall_snapshot

Purpose: Provides a high-level snapshot of performance for each client and offer over standard reporting periods (e.g., L7D, L30D, MTD). This is useful for dashboards and executive summaries.

Grain: client_name, client_offer, period_type

Column Name Data Type Mode Constraints / Notes Description
client_name STRING NULLABLE PK Name of the client.
client_offer STRING NULLABLE PK Specific offer (or 'All Offers').
period_type STRING NULLABLE PK The period this snapshot represents (e.g., 'L7D', 'L30D', 'MTD', 'LastMonth').
period_start_date DATE NULLABLE Start date of the snapshot period.
period_end_date DATE NULLABLE End date of the snapshot period.
ad_network STRING NULLABLE Advertising network.
total_spend FLOAT NULLABLE Total amount spent in the period.
total_impressions INTEGER NULLABLE Total impressions in the period.
total_clicks INTEGER NULLABLE Total clicks in the period.
total_reach INTEGER NULLABLE Total unique users reached in the period.
total_purchases INTEGER NULLABLE Total purchase conversions in the period.
total_leads INTEGER NULLABLE Total lead conversions in the period.
total_registrations INTEGER NULLABLE Total registration conversions in the period.
total_conversion_value FLOAT NULLABLE Total monetary value from conversions in the period.
roas FLOAT NULLABLE ROAS for the period.
cpc FLOAT NULLABLE CPC for the period.
cpm FLOAT NULLABLE CPM for the period.
ctr FLOAT NULLABLE CTR for the period.
cpa_purchase FLOAT NULLABLE Cost Per Purchase for the period.
cpa_lead FLOAT NULLABLE Cost Per Lead for the period.
cpa_registration FLOAT NULLABLE Cost Per Registration for the period.
frequency FLOAT NULLABLE Average frequency for the period.
number_of_active_campaigns INTEGER NULLABLE Total active campaigns in the period.
number_of_active_adsets INTEGER NULLABLE Total active ad sets in the period.
number_of_active_ads INTEGER NULLABLE Total active ads in the period.
top_roas_campaign_name STRING NULLABLE Name of the campaign with the highest ROAS.
top_roas_campaign_value FLOAT NULLABLE Highest ROAS value.
lowest_cpl_campaign_name STRING NULLABLE Name of the campaign with the lowest CPL.
lowest_cpl_campaign_value FLOAT NULLABLE Lowest CPL value.
highest_ctr_campaign_name STRING NULLABLE Name of the campaign with the highest CTR.
highest_ctr_campaign_value FLOAT NULLABLE Highest CTR value.
last_updated TIMESTAMP NULLABLE Timestamp of when the record was last updated.

Source: Aggregated from v1_b2s_all_acquisition_production_report or the periodic summary tables for standard periods.
